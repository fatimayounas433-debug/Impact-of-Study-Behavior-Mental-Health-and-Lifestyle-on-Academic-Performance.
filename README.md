📊 Impact of Study Behavior, Mental Health & Lifestyle on Academic Performance
About This Project
I've always been curious about what really affects a student's grades. Is it just hard work? Or do things like sleep, stress, and daily habits matter just as much?

So I decided to find out — using data.

This project explores how study behavior, mental health, and lifestyle influence academic performance among university students. I used SPSS to run correlations and regression analysis to uncover what truly predicts GPA.

Why This Matters
Grades aren't just about intelligence. They're about the whole person — how they study, how they feel, and how they live. Understanding this can help students, teachers, and even employers create better support systems.

Tools Used
SPSS (data cleaning, correlation, multiple regression)
Key Findings
Study quality matters more than study quantity
Mental health plays a significant role in performance
Lifestyle habits (sleep, exercise) are strong predictors
